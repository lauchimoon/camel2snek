# camel2snek
Turn text from this
```
someText
AlsoSomeText
```

To this
```
some_text
also_some_text
```

## Usage
```sh
$ git clone https://github.com/lauchimoon/camel2snek.git
$ cd camel2snek/
$ python3 {PROGRAM_NAME} <text>|--file <filename>
```

## Caveats
This
```
HELLO
```

Gets turned into
```
h_e_l_l_o
```
